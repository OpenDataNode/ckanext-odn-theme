ckanext-odn-theme
-------

ODN theme for CKAN

Installation
-------

enable plugin:
add odn_theme to ckan.plugins in your production.ini

set in .ini:
* ckan.site_title = COMSODE - Open Data Node 
* ckan.site_logo = /base/images/odnlogo.png
